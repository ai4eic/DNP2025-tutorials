How to Use#

  1. Install Dependencies:

    pip install -r ../requirements.txt

  2. Launch Jupyter:

    jupyter lab

  3. Open Notebooks: Navigate to the desired notebook and run cells sequentially.

Contributing#

To contribute a notebook:

  1. Follow the naming convention: ##_descriptive_name.ipynb

  2. Include:

    • Title and overview

    • Learning objectives

    • Required imports

    • Clear explanations

    • Working code cells

    • Visualizations

    • Exercises (optional)

    • Summary

  3. Test thoroughly before submitting

  4. See CONTRIBUTING.md for details

Notebook Template#

A template notebook is provided: notebook_template.ipynb

Copy and customize for new tutorials:

cp notebook_template.ipynb your_new_notebook.ipynb

Support#
